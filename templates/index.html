Service Account Setup

Please follow the instructions below to create your service_account.json file:

  1. Visit the Google Cloud Console and create or select a project.
  2. Navigate to APIs & Services > Library and enable the required APIs (e.g., Google Drive API, Google Sheets API).
  3. Go to APIs & Services > Credentials and click on Create Credentials > Service account.
  4. Fill in the required details and click Create.
  5. Once the service account is created, click on it, go to the Keys tab, and click Add Key > Create new key.
  6. Select JSON as the key type and click Create. This will download the service_account.json file.
  7. Save this file in the root directory of your project (or in a secure location) and ensure your code references the correct path.
